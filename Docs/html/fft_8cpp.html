void reorder() Selects every Nth value and places it in front of the array

Iterates over an array for the given size from the given start point Then it swaps those values with those at front. The functionality is similar to what std::slice does and it is based on it. The only difference between reorder and slice is that reorder uses the original array and doesn't throw it away.
