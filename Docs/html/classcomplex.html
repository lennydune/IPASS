Detailed Description

class complex The class to implement complex numbers

The c++-11 implementation of complex numbers doesn't play nice with hwlib. Therefore I copied the parts I needed from the c++-11 implementation over here. There are still some problems however, something with operator new() and operator delete() not being defined.
